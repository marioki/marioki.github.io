Wordle Clone

This is, of course, a Wordle game clone. I play Wordle every day, usually while drinking my morning coffee, so I had a lot of fun figuring out how to make it. Currently, it is set up to pull a new word every time you load the page. So, if you fail to guess the word, just hit refresh and start a new game!
