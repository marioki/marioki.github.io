It's an iOS calculator clone.

I started this project as part of the begginer front end course I'm following. It comes after the intro to JavaScript section. For simplicity it only allows integers and the percentage operation is missing.

Its also missing +/- button to change the number sign to negativo or positive. I may continue to improve and add new things to it but I'm happy with this version for now.

For some reason it does not work well on mobile browsers. It feels laggy on a mobile phone. After some digging it looks to be something with the use of buttons and or click events. Did a small experiemnt with touch/touchend events and using touchend seems to fix the issue. I gues some html components don't play well on mobile.
